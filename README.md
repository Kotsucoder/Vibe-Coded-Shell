# Vibe-Coded Shell

![A demonstration of the Vibe-Coded Shell in action.](hero.jpg)

This project is a POSIX-compliant shell built in Python. What makes it special is its origin: every line of code was generated by Google's Gemini, with a human developer (me!) acting as the guide. 

This repository is a public journal of this experiment, created for two main reasons.

## Project Goals

1.  **Putting AI to the Test**: Just how capable are modern AI tools at writing functional, compliant systems software? This shell serves as a tangible answer, showcasing the strengths and weaknesses of AI-driven development in a real-world scenario.

2.  **Sharpening My Code Review Skills**: The second purpose is a personal development goal. By tasking an AI to write the code, I've created a unique opportunity to hone my own code review abilities. My work here is to rigorously inspect the AI's output for correctness, efficiency, and security, documenting my findings and refining the code.
